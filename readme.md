Chattery:

A social media application developed using flutter and firebase.
Currently available functionality:

1. User authentication : login and sign up.
2. One-on-one real time chat, with Online status display
3. Posting images, and viewing friends; posts.
4. Like and comment on friends' posts!
5. Edit profile: Add more about yourself
6. Search for friends!
